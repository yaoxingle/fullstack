## 如何本地运行

### 快速开始

建议`docsify-cli`全局安装，有助于在本地初始化和预览网站。

```bash
npm i docsify-cli -g
```

### 预览您的网站

运行本地服务器`docsify serve`。您可以在浏览器中预览您的网站`http://localhost:3000`。

```bash
docsify serve docs
```

有关 的更多用例`docsify-cli`，请转到[docsify-cli 文档](https://github.com/docsifyjs/docsify-cli)。
