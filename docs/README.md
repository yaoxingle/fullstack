# 项目介绍

java 全栈技术积累文档,基于 [docsify](https://docsify.js.org/#/) 搭建。 访问进入[github pages](https://yaoxingle.github.io/fullstack/#/)

## 项目运行方式

* node
<br>

    安装 node 环境

    ```js
    node app
    ```

* docsify-cli  **推荐**
<br>

    1、下载 docsify-cli 包

    ```js
    npm i docsify-cli -g
    ```
    2、然后进入 **public** 文件夹，执行

    ```js
    docsify serve
    ```
