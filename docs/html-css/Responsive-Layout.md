**移动端app的存在形式一共有3类**

1.原生应用:Native app 性能更流畅、功能更强大、可以直接调用硬件接口。必须经常骚扰用户去更新

2.Web应用:Web App 性能相对较差（依赖于浏览器的持之），无法直接调用硬件，无需通知更新，部署方便，权限小，用户信任度高。

3.混合应用：Hybrid App
