 解压密码：zdfans 

###### 1、先不要运行软件，将注册机复制到软件安装目录下运行，点击Patch弹出提示即可 

 ![img](images/202001111127508725.jpg) 

###### 2、 运行软件，点击Registration注册 

 ![img](images/202001111127568741.jpg) 

###### 3、 选择对应的软件及语言，然后点击Generate生成序列号 

 ![img](images/202001111128039381.jpg) 

###### 4、 选择Manual Activation手动激活 

 ![img](images/202001111128096119.jpg) 

###### 5、 将Request Code复制到注册机中，然后点击Generate生成激活码，复制到软件中激活 

 ![img](images/202001111128152142.jpg)
